## 三：系统学习

### 1. 自动补全功能

- `__name__ == "__main__"`
- for循环补全---->"iter"
- 后缀补全---->"name."

**查找功能**

- **基本搜索**（文件内搜索）：`ctrl + F`
- **超级搜索**（搜索目录）：`ctrl + shift + F`
  - 大小写
  - 搜索文件后缀
  - 搜索文件目录
- **文件搜索**（搜索文件/符号）：`shift + shift`
  - pull
  - push
  - 快速定位符号（类、函数、方法）

### 2. 版本控制

**功能**：配置项目解释器，不同项目对应不同的库

**设置位置**：设置---->project interpreter

### 5. 快速复制删除行

- **换行**：`shift + enter`
